缓存导出工具 / Cache Export Tool

这是网站维护者使用的备用工具，普通玩家不需要使用。正常情况下，GitHub Actions 会自动更新缓存文件。

This is a backup tool for site maintainers. Regular players do not need to use it. Normally, GitHub Actions updates the cache files automatically.

这个工具只通过公开 Firebase 客户端 SDK 读取数据并生成可下载 JSON，不会写入 GitHub，也不包含 Firebase Admin 私钥或 GitHub Token。

This tool only reads data through the public Firebase client SDK and generates downloadable JSON. It does not write to GitHub and does not contain Firebase Admin credentials or GitHub tokens.
