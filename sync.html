这是网站维护者使用的备用工具，普通玩家不需要使用。正常情况下，留言和分数会自动提交；只有在需要手动处理 ZIZI-SYNC 同步码时才会用到这里。

This is a backup tool for site maintainers. Regular players do not need to use it. Comments and scores are normally submitted automatically; this page is only for manually handling ZIZI-SYNC codes when needed.
