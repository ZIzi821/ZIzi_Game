# ZIzi_Game

网页游戏合集仓库，使用 GitHub Pages 从 `main` 分支根目录发布。

网站主页：

https://zizi821.github.io/ZIzi_Game/

## Games

- [Aetherfall Protocol](https://zizi821.github.io/ZIzi_Game/aetherfall-protocol/)
- [Starline Defense](https://zizi821.github.io/ZIzi_Game/starline-defense/)
- [Pacman Odyssey](https://zizi821.github.io/ZIzi_Game/pacman-odyssey/)

## Folder Layout

```text
/
  index.html
  README.md
  aetherfall-protocol/
  starline-defense/
  pacman-odyssey/
```

以后新增游戏时，直接在根目录创建新的游戏文件夹，并在 `index.html` 和 `README.md` 里加入口链接。
