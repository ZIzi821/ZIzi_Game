# ZIzi_Game

ZIzi 的游戏合集仓库，使用 GitHub Pages 从 `main` 分支根目录发布。

网站主页：
https://zizi821.github.io/ZIzi_Game/

更新日志：
https://zizi821.github.io/ZIzi_Game/updates.html

## Games

- [Starfall / 星陨](https://zizi821.github.io/ZIzi_Game/aetherfall-protocol/)：能量核心生存射击，程序化波次与升级选择。
- [Sentinel / 哨兵防线](https://zizi821.github.io/ZIzi_Game/starline-defense/)：多路线塔防，部署单位守住核心防线。
- [Chomp / 大口吃](https://zizi821.github.io/ZIzi_Game/pacman-odyssey/)：浏览器版霓虹迷宫吃豆游戏。

## Source Projects

- `pacman_game/`：Chomp / 大口吃 的 Pygame 源码版本，包含图片、音效、音乐和运行脚本。

## Folder Layout

```text
/
  index.html
  README.md
  updates.html
  aetherfall-protocol/
  starline-defense/
  pacman-odyssey/
  pacman_game/
```

新增游戏时，在根目录创建独立文件夹，并在 `index.html` 和 `README.md` 里添加入口链接。
